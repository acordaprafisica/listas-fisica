# Listas de Exercícios — Acorda pra Física

Coleção de listas de exercícios de física em HTML, prontas para impressão e uso em aula.

## Estrutura

O repositório é organizado por área de conteúdo, seguindo `docs/mapa_conteudos_fisica_enem_v2.md`:

```
/listas
  /01-eletricidade
    /eletrostatica       — listas de aprofundamento por subtema
    /revisoes             — listas de revisão que misturam subtemas da área
  /02-termologia
    /calorimetria
    /revisoes
  /03-ondulatoria
    /ondulatoria-geral
    /revisoes
  /05-optica-geometrica   — listas que já cobrem vários subtemas da área de uma vez
/experimentais            — materiais fora do formato padrão de lista impressa
/docs                     — mapa de conteúdos, metodologia de geração e relatórios de auditoria
```

As pastas `04-mecanica` e `06-fisica-moderna` (e as demais subpastas ainda
vazias) nascem quando as primeiras listas desses temas forem criadas — ver
`docs/relatorio_cobertura.md` para o backlog do que falta.
